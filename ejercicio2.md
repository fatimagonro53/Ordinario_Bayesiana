# Ejercicio 2: Regla del complemento

## Enunciado

¿Cuál es la probabilidad de NO obtener un 6 al lanzar un dado?

## Paso 1

P(6) = 1/6

## Paso 2

Aplicar la regla del complemento:

P(no 6) = 1 - P(6)

## Resultado

P(no 6) = 1 - 1/6 = 5/6

P(no 6) = 0.8333

## Código Python

```python
p6 = 1/6

p_no6 = 1 - p6

print(p_no6)
```
