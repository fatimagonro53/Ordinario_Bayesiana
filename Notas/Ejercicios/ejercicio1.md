# Ejercicio 1: Probabilidad Simple

## Enunciado

Se lanza un dado justo de seis caras.

¿Cuál es la probabilidad de obtener un 4?

## Paso 1

Casos favorables:

1

## Paso 2

Casos posibles:

6

## Paso 3

Aplicar la fórmula:

P(A) = Casos favorables / Casos posibles

P(A) = 1/6

## Resultado

P(A) = 0.1667

## Código Python

```python
favorables = 1
posibles = 6

probabilidad = favorables / posibles

print(probabilidad)
