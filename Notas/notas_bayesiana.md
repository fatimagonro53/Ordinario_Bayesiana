# Notas de Probabilidad Bayesiana

## Probabilidad

P(A) = Casos favorables / Casos posibles

## Complemento

P(Ac) = 1 - P(A)

## Probabilidad Condicional

P(A|B) = P(A ∩ B) / P(B)

## Independencia

P(A ∩ B) = P(A)P(B)

## Teorema de Bayes

P(B|A) = [P(A|B)P(B)] / P(A)

## Esperanza

E(X) = ΣxP(x)

## Varianza

Var(X) = E(X²) - [E(X)]²

## Distribución Binomial

P(X=k)=C(n,k)p^k(1-p)^(n-k)

## Distribución Normal

Z=(x-μ)/σ
