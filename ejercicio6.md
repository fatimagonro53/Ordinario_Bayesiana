# Ejercicio 6: Esperanza Matemática

## Enunciado

Se lanza un dado justo. Calcular la esperanza matemática.

## Fórmula

E(X) = Σ x·P(x)

## Cálculo

E(X) = (1+2+3+4+5+6)/6

E(X) = 3.5

## Resultado

La esperanza matemática es 3.5.

## Código Python

```python
valores = [1,2,3,4,5,6]

esperanza = sum(valores)/len(valores)

print(esperanza)
```
